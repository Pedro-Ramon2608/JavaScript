# JavaScript

🚀 Exercícios de JavaScript (Curso em Vídeo)

Este repositório contém todos os desafios e exercícios práticos desenvolvidos durante o curso de JavaScript do portal Curso em Vídeo, ministrado pelo professor Gustavo Guanabara.

#
💻 Sobre o Projeto

O foco principal destas práticas foi entender como o JavaScript dá vida às páginas web, saindo da lógica pura de terminal para a criação de interfaces dinâmicas e interativas.

#
🛠️ Tecnologias Utilizadas

* JavaScript (ES6+)

* HTML5

* CSS3

#
📚 O que foi praticado:

🔹 Fundamentos e Lógica

* Variáveis e tipos de dados.

* Operadores aritméticos, lógicos e relacionais.

* Condicionais (if/else e switch).

* Estruturas de repetição (while e for).

🔹 Manipulação do DOM (Document Object Model)

* Eventos de mouse e teclado.

* Seleção de elementos (getElementById, querySelector).

* Alteração dinâmica de estilos e conteúdos via JS.

🔹 Projetos de Destaque

Durante o curso, foram desenvolvidos pequenos projetos práticos que aplicam todo o conhecimento:

1. Verificador de Horário: Altera a imagem e a cor do fundo da página de acordo com a hora do sistema.

2. Verificador de Idade: Calcula a idade e exibe uma imagem correspondente ao perfil detectado.

3. Contador Inteligente: Realiza contagens personalizadas com diferentes passos.

4. Tabuada Dinâmica: Gera tabuadas instantaneamente para qualquer número.

5. Analisador de Números: Um sistema que recebe vários valores, armazena em um array e entrega estatísticas como soma e média.

📂 Como visualizar

Como este repositório utiliza tecnologias Front-end (HTML/CSS/JS), você pode visualizar os exercícios apenas abrindo os arquivos .html no seu navegador ou utilizando a extensão Live Server no VS Code.
